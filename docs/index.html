BenBotOS™

Features

BenbotOS™ features are built for automation. From semi-automatic, like the Smart Tunneling and Bridging app, which automatically takes care of the action and finishes, to fully automatic applications like the tree farm application, that keep going forever until stopped, there is no shortage of automation.

Ease of use

Installing

The install process for BenBotOS™ is very simple. If you get your hand on an install disk, it can be installed on any robot running openOS by running the program attached. It even runs on top of openOS for maximum speed and reliability.

General Usage

 BenBotOS™ is itself just an addition to openOS. Meaning that BenBotOS™ behaves exactly like openOS. All programs on BenBotOS™ do not require any complicated arguments, and instead only require simple questions instead.

Open Source

BenBotOS™ is completly open source. You can even contribute to it!
