BenBotOS™

Features

  • Completely Automatic Applications: These are applications that once you start, they never end. They fulfill a certain goal without any intervention whatsoever.
  • Semi-Automatic Applications: These applications are started and may ask a few questions about the task before starting, then the tasks are automatically completed without any intervention and automatically exit.

Ease of use

Installing

The install process for BenBotOS™ is very simple. If you get your hand on an install floppy, it can be installed on any robot running OpenOS by running the program attached. BenBotOS™ is designed to run on top of OpenOS.

General Usage

BenBotOS™ is an addition to OpenOS, meaning that BenBotOS™ behaves exactly like OpenOS. All programs on BenBotOS™ do not require any complicated arguments, and instead only require simple questions if needed.

Open Source

BenBotOS™ is completely open-source. You can even contribute to it!
